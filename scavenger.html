How It Works

A secret message hiding in plain arithmetic

The idea is simple: each letter in a secret sentence gets converted into an arithmetic problem. Solve the problems, decode the answers back into letters, and the hidden message is revealed. The tools here offer different ways to do that encoding — some map each letter to a single number, others use a coordinate position on a grid — but the principle is the same in every case.

1 Type a secret sentence into whichever tool you choose.
2 Choose which arithmetic operations to use — addition, subtraction, multiplication, or division.
3 A printable worksheet is generated with one or more arithmetic problems per letter.
4 Solve the problems, decode the answers back into letters, and reveal the hidden message.
The Tools

Two ways to hide a message

The Scavenger Hunt Generator assigns each letter a number — forward (A=1…Z=26), reverse (A=26…Z=1), or shuffled randomly. Each letter in the sentence becomes one arithmetic problem whose answer is that letter’s number. A key on the worksheet lets the solver look up their answer and find the letter.

The Cartesian Scavenger Hunt takes a different approach. Letters are placed randomly in a 10×10 grid, and each letter’s position is given as a coordinate pair (x, y). Each letter in the sentence becomes two arithmetic problems — one for x, one for y. The solver finds the correct cell in the grid and reads off the letter. Both tools produce a printable worksheet ready to hand to a child.
