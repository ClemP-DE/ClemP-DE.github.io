How It Works

A secret message hiding in plain arithmetic

Each letter of the alphabet maps to a number — A=1, B=2, all the way to Z=26, or in reverse (A=26, Z=1) for an extra layer of challenge. Type in any sentence, choose which arithmetic operations to use, and the tool generates a printable worksheet.

1 Type a secret sentence into the generator — anything from "Hello" to a full phrase.
2 Choose which operations to use: addition, subtraction, multiplication, division, or any combination.
3 The tool generates a printable worksheet — one arithmetic problem per letter, each with an answer that maps back to a letter.
4 Solve the problems, look up each answer in the mapping table at the bottom of the sheet, and the secret sentence is revealed.
