Conjecture

The Collatz Conjecture

Take any positive integer. If even, halve it. If odd, triple it and add one. Repeat. The conjecture says you will always eventually reach 1. Nobody has proven this. Nobody has disproven it. This project explores it through visualisation — treating the sequences as a data set and using interactive tools to build intuition.

Interactive Tools

Seven tools for exploring the conjecture

Each tool grew out of a question raised by the previous one. Open them in Chrome for best results. You can also download the HTML files and upload them into an LLM to extend them further.

01 Trajectory Diagram Plots the sequence for any seed as a path through the phase diagram, with colour-coded arrows showing which rule fired at each step. → 02 Sequence Table Shows the full colour-coded sequence for every seed in a range — red for odd values, blue for even, green for 1. Sortable by seed, steps, or peak. → 03 Steps Histogram Plots the distribution of stopping times across a seed range. Reveals the curious two-peak structure and how the peaks drift as the range grows. → 04 Odd/Even Step Counter For each seed, counts how many steps applied the odd rule (3n+1) versus the even rule (n÷2) and shows the ratio between them. → 05 Steps vs Even/Odd Ratio Scatter plot of total steps against the even/odd ratio. Shows how the ratio narrows toward ~1.7 as sequences grow longer. → 06 Generalized Explorer Replaces the standard rule with (k+1)n+1 and n÷k. Explore which seeds still converge to 1, which enter cycles, and which diverge as k changes. → 07 Vector Path Length Treats consecutive points in the phase diagram as vectors and computes the total Euclidean distance travelled before reaching 1. →
